
# คู่มือการใช้งาน Calculator-teerasej

สร้างไว้ใช้บวกเลขชิลล์ๆ

สร้างตอนเรียน Node.js กับพล

- สามารถบวกเลขได้
- ลบได้ 
- คูณได้
- แต่หารยังไม่ได้ทำ

## วิธีใช้งาน

วิธี require module 

```js
const calculator = require('calculator-teerasej');
```

### วิธีการบวก

สำหรับการบวก ให้ใช้ function `plus()`

```js
calculator.plus(1,2) // 3
```
